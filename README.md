# Stellar Tip Jar - Level 1 White Belt

## Description
A simple dApp built on the Stellar Testnet that allows users to connect their Freighter wallet, check their XLM balance, and send XLM as a tip. 

## Setup Instructions
1. Clone this repository or simply download the `index.html` file.
2. Open `index.html` in any modern web browser (using VS Code Live Server is recommended).
3. Ensure you have the Freighter wallet extension installed, active, and switched to the Stellar Testnet.
